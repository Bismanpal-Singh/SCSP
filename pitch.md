# CriticalMat 2.0 — Stage Pitch Script

## Live Input Line (type this on stage)
"Find a solid-state, manufacturable replacement material for a defense
component that avoids Chinese-controlled rare earths and radioactive elements."

## 60-Second Script

**(0-10s)** "Critical material replacement today is slow because researchers
must manually connect mission needs, databases, safety constraints, supply-chain
risk, and experiment planning. A process that takes months."

**(10-25s)** "CriticalMat parses the mission need into scientific and strategic
constraints, then searches Materials Project — 150,000 real computed materials —
as a virtual lab. It screens against safety, radioactivity, and supply chain
risk simultaneously."

**(25-40s)** "It rejects unsafe candidates with clear reasons, ranks eligible
materials into families, and identifies what is still scientifically unknown
about each one."

**(40-55s)** "Instead of returning one answer, it builds a lab-ready portfolio:
what to test first, what to keep as backup, what uncertainty each test reduces,
and what experiment to run next. This is pre-lab planning automation."

**(55-60s)** "This is an autonomous pre-lab planning agent for national-security
materials: not just discovery, but decision-ready experimentation."

## Close Line
"CriticalMat does not just find a replacement material.
It creates the test plan a real lab can act on tomorrow."
